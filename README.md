# Socials

[![CI](https://github.com/anton415/socials/actions/workflows/ci.yml/badge.svg?branch=main)](https://github.com/anton415/socials/actions/workflows/ci.yml)

Учебный REST API на Spring Boot для практики проектирования бэкенда социальной сети:
пользователи, посты, комментарии, подписки, лента.
Цель — понятный «скелет» проекта с хорошими практиками (слои, миграции, CI, документация API).

## Стэк технологий

- Java 17+ (Spring Boot 4.x требует Java 17 или новее)
- PostgreSQL (локально или через Docker)
- Git
- Maven

## Команды

Сборка проекта:

```bash
./mvnw clean package
```

Запуск тестов:

```bash
./mvnw test
```
